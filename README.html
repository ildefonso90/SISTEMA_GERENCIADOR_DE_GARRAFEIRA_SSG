🍷 Garrafeira - Sistema de Gestão

O Garrafeira é um sistema de gerenciamento de bebidas que facilita o controle de estoque, vendas e clientes.

📌 Funcionalidades

  • 🔑 Login seguro (Administrador e Atendente)
  • 📦 Controle de estoque completo
  • 🛒 Registro e histórico de vendas
  • 👥 Gerenciamento de clientes e fornecedores
  • 📊 Relatórios detalhados

🚀 Como Executar

Para rodar o projeto, siga estes passos:

1. Clone o repositório:
git clone https://github.com/seuusuario/garrafeira.git

2. Abra no Visual Studio e compile.
3. Configure o banco de dados se necessário.
4. Execute o projeto e comece a usar! 🚀
